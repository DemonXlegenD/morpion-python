# Morpion en Python

## Description

Ce programme en Python implémente le jeu du morpion (ou Tic-tac-toe) jouable en console. Le jeu permet à deux joueurs de s'affronter sur le même appareil. Le premier joueur joue avec les "X" et le deuxième joueur avec les "O". Le but du jeu est d'aligner trois symboles identiques soit horizontalement, verticalement ou en diagonale.

## Fonctionnalités

- Interface console pour jouer au morpion.
- Possibilité de jouer à deux joueurs sur le même appareil.
- Détection automatique du gagnant ou du match nul.
- Possibilité de recommencer une partie après la fin d'une partie.

## Installation

1. Clonez ce dépôt sur votre machine locale.
2. Assurez-vous d'avoir Python installé sur votre machine. Si ce n'est pas le cas, téléchargez et installez Python à partir de [python.org](https://www.python.org/).
3. Naviguez vers le répertoire contenant le fichier `morpion.py`.
4. Exécutez le programme en ouvrant le fichier avec python
